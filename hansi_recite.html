🎤 마이크 음성 인식 사용 방법
현재 claude.ai 미리보기 창에서는 브라우저 보안 정책(iframe 샌드박스)으로 인해 마이크가 차단됩니다.
파일을 다운로드한 후 아래 방법으로 실행하면 마이크가 정상 작동합니다.
  • 파일을 다운로드하여 컴퓨터에 저장합니다
  • 터미널(명령 프롬프트)에서 파일이 있는 폴더로 이동 후 python -m http.server 8080 실행
  • Chrome에서 http://localhost:8080/hansi_recite.html 접속 → 마이크 허용
또는 VS Code의 Live Server 확장 프로그램을 사용하면 더 간편합니다.
